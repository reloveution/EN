PAST SIMPLE
Обычное завершённое действие в прошлом (когда — известно или неважно). Регулярные действия в прошлом.

st act
S + V2/Had + O. <|> S + Was/Were + Adj/Place.
        Учитель объяснил правило.
        
        
st pas
O + Was/Were + V3 + (by)S.
        Правило было объяснено учителем.
        
        
qt act
Did + S + V1 + O? <|> Had + S + O? <|> Was/Were + S + Adj/Place?
        Учитель объяснил правило?
        
        
qt pas
Was/Were + O + V3 + (by)S?
        Правило было объяснено учителем?
        

        
ng act
S + Did Not + V + O. <|> S + Had Not + O. <|>  S + Was/Were Not + Adj/Place.
        Учитель не объяснил правило.
        
        
ng pas
O + Was/Were Not + V3 + (by)S.
        Правило не было объяснено учителем.

FUTURE SIMPLE
Действие в будущем (решение, предсказание, обещание). Will или going to (намерение, план).

st act
S + Will + V + O. <|> S + Will be + Adj/Place. <|> S + Am/Is/Are + going to + V + O. <|> S + Am/Is/Are + going to be + Adj/Place.
        Учитель объяснит правило на следующем уроке. / Учитель собирается объяснить правило на следующем уроке.

        
st pas
O + Will be + V3 + (by)S. <|> O + Am/Is/Are + going to be + V3 + (by)S.
        Правило будет объяснено учителем на следующем уроке. / Правило собираются объяснить на следующем уроке.

        
qt act
Will + S + V + O? <|> Will + S + be + Adj/Place? <|> Am/Is/Are + S + going to + V + O? <|> Am/Is/Are + S + going to be + Adj/Place?
        Учитель объяснит правило на следующем уроке? / Учитель собирается объяснить правило?

        
qt pas
Will + O + be + V3 + (by)S? <|> Am/Is/Are + O + going to be + V3 + (by)S?
        Правило будет объяснено учителем на следующем уроке? / Правило собираются объяснить?

        
ng act
S + Will Not + V + O. <|> S + Will Not be + Adj/Place. <|> S + Am/Is/Are Not + going to + V + O. <|> S + Am/Is/Are Not + going to be + Adj/Place.
        Учитель не объяснит правило на следующем уроке. / Учитель не собирается объяснять правило.

        
ng pas
O + Will Not be + V3 + (by)S. <|> O + Am/Is/Are Not + going to be + V3 + (by)S.
        Правило не будет объяснено учителем на следующем уроке. / Правило не собираются объяснять.
