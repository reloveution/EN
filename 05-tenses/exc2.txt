-----------EXCERSIZE---------


PAST SIMPLE
Обычное завершённое действие в прошлом (когда — известно или неважно). Регулярные действия в прошлом.

st act
S + V2/Had + O. <|> S + Was/Were + Adj/Place.
    Вчера я купил очки.
    I bought glasses yesterday.
        

st pas
O + Was/Were + V3 + (by)S.
    Очки были куплены вчера.
    The glasses were bought by me yesterday.
        

qt act
Did + S + V1 + O? <|> Had + S + O? <|> Was/Were + S + Adj/Place?
    Я купил вчера очки?
    Did I buy glasses yesterday?

qt pas
Was/Were + O + V3 + (by)S?
    Очки были куплены вчера?
    Were glasses bought by me yesterday?

ng act
S + Did Not + V + O. <|> S + Had Not + O. <|>  S + Was/Were Not + Adj/Place.
        Я не купил вчера очки.
        I didn't buy glasses yesterday.

ng pas
O + Was/Were Not + V3 + (by)S.
        Очки были не куплены.
        The glasses weren't bought.


        


PAST CONTINUOUS
Длительное действие в прошлом (в процессе в какой-то момент). Часто — фон для другого действия или незавершённость.

st act
S + Was/Were + V-ing + O. <|> S + Was/Were + being + Adj.
        Я покупал очки вчера.
        I was buying glasses.

st pas
O + Was/Were + being + V3 + (by)S.
    The glasses were being bought by me.
    Очки покупались мной вчера.
        

qt act
Was/Were + S + V-ing + O? <|> Was/Were + S + being + Adj?
        Was I buying the glasses?
        Я покупала очки вчера вечером?

qt pas
Was/Were + O + being + V3 + (by)S?
    Очки покупались вчера вечером?
    Were glasses being bought by me last night?
        

ng act
S + Was/Were Not + V-ing + O. <|> S + Was/Were Not + being + Adj.
    I wasn't buying glasses last night.
    Я не занимался покупкой очков вчера вечером.

ng pas
O + Was/Were Not + being + V3 + (by)S.
    The glasses weren't being bought by consumers last night.
    Очки не покупались(продавались) потребителями вчера.


PAST PERFECT
Действие уже завершилось до другого действия или момента в прошлом («предпрошедшее»).

st act
S + Had + V3 + O. <|> S + Had been + Adj/Place.
    Я купил очки до закрытия магазина.
    I had bought glasses before a store was closed.

        

st pas
O + Had been + V3 + (by)S.
    Очки были раскуплены до закрытия магазина.
    The glasses had been bought before a store closed
        

qt act
Had + S + V3 + O? <|> Had + S + been + Adj/Place?
    Я купил учки до закрытия магазина?
    Had I bought glasses before a store was closed?
        

qt pas
Had + O + been + V3 + (by)S?
        Очки были куплены до закрытия магазина?
        Had glasses been bought before a store was closed?

ng act
S + Had Not + V3 + O. <|> S + Had Not been + Adj/Place.
        I hadn't bought glasses before a store was closed.
        Я не купил очки до того как магазин был закрыт.
ng pas
O + Had Not been + V3 + (by)S.
    Очки не были куплены до закрытия магазина.
    The glasses hadn't been bought before a store was closed.

FUTURE SIMPLE

S + Will + V + O. <|> S + Will be + Adj/Place. <|> S + Am/Is/Are + going to + V + O. <|> S + Am/Is/Are + going to be + Adj/Place.
