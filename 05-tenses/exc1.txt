PAST SIMPLE
Обычное завершённое действие в прошлом (когда — известно или неважно). Регулярные действия в прошлом.

st act
S + V2/Had + O. <|> S + Was/Were + Adj/Place.
        Менеджер подписал контракт.
        The manager signed the contract.
        
st pas
O + Was/Were + V3 + (by)S.
        Контракт был подписан менеджером.
        The contract was signed by the manager.
        
qt act
Did + S + V1 + O? <|> Had + S + O? <|> Was/Were + S + Adj/Place?
        Менеджер подписал контракт?
        Did the manager sign the contract?
        
qt pas
Was/Were + O + V3 + (by)S?
        Контракт был подписан менеджером?
        Was the contract signed by the manager?

        
ng act
S + Did Not + V + O. <|> S + Had Not + O. <|>  S + Was/Were Not + Adj/Place.
        Менеджер не подписал контракт.
        The manager didn't sign the contract.
        
ng pas
O + Was/Were Not + V3 + (by)S.
        Контракт не был подписан менеджером.
        The contract wasn't signed by the manager.

FUTURE SIMPLE

O + Will be + V3 + (by)S. <|> O + Am/Is/Are + going to be + V3 + (by)S.
        Контракт будет подписан менеджером в следующем месяце. / Контракт собираются подписать в следующем месяце.

        
qt act
Will + S + V + O? <|> Will + S + be + Adj/Place? <|> Am/Is/Are + S + going to + V + O? <|> Am/Is/Are + S + going to be + Adj/Place?
        Менеджер подпишет контракт в следующем месяце? / Менеджер собирается подписать контракт?

        
qt pas
Will + O + be + V3 + (by)S? <|> Am/Is/Are + O + going to be + V3 + (by)S?
        Контракт будет подписан менеджером в следующем месяце? / Контракт собираются подписать?

        
ng act
